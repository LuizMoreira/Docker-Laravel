<p align="center">Docker-laravel</p>

## Ambiente Laravel utilizando o docker-compose com:
- Nginx
- PHP-FPM
- Redis
- MySQL

## Instalação:
Clone o projeto e rode pelo seguinte comando:

```
sudo git clone https://github.com/LuizMoreira/Docker-Laravel.git
cd Docker-Laravel
docker-compose up -d
```
